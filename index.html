Notable
Notable is a prototype for inline annotation. Based off Tabletop.js, it is powered by Google Spreadsheets. Use the format in this spreadsheet. After you have created a spreadsheet of your own with the headers in the sample, go to File -> Publish to the web... and copy the link. Insert the link in the box below and Notable will help you add annonations to your story.
